Western Australian coordinate reference systems

https://linked.data.gov.au/def/wa-coordinate-reference-systems

Metadata

DefinitionThe coordinate reference systems (geographic and projected) used and referred to within Western Australia. It includes historical and current reference systems, Universal Transverse Mercator (UTM) zones, and, where applicable, notations based on the EPSG registry (https://epsg.io/). Some concepts are included to account for legacy data/maps with limited information.
